# Penguen ile Dinimi Öğreniyorum 🐧

Bu proje, çocukların ve gençlerin namaz rekatlarını ve kısa surelerin okunuşları ile anlamlarını interaktif bir şekilde öğrenmesini sağlayan eğlenceli bir Python konsol uygulamasıdır.

## 🚀 Özellikler
- ASCII Art ile sevimli penguen karşılama ekranı
- Detaylı namaz rekatları ve ek bilgiler
- Sözlük (dict) yapısı kullanılarak dinamik olarak listelenen ve sorgulanan sureler (Fatiha'dan Nas suresine kadar)
- Kolay menü navigasyonu ve döngü sistemi

## 📦 Kurulum ve Çalıştırma
Projeyi bilgisayarınızda çalıştırmak için Python 3'ün kurulu olması yeterlidir:

```bash
python pengueniledinimiogreniyorum-demo.py
```
